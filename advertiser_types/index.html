    aiosubgram.types.advertiser.UserParameters

    Bases: SubgramObject

    Параметры таргетинга для заказа. Влияют на итоговую стоимость подписчика (коэффициенты).

    Attributes

    ages class-attribute instance-attribute 
    ages: Optional[List[int]] = None

    ID категорий возраста. Коэффициенты: <10 (x1.2), 11-13 (x1.2), 14-15 (x1.5), 16-17 (x1.5), 18-24 (x2.0), >25 (x2.5).

    cities class-attribute instance-attribute 
    cities: Optional[List[int]] = None

    ID городов (из /filters). Коэффициент: x2.5

    countries class-attribute instance-attribute 
    countries: Optional[List[int]] = None

    ID стран (из /filters). Коэффициент: x2.5

    devicesos class-attribute instance-attribute 
    devicesos: Optional[List[int]] = None

    ID операционных систем. Коэффициент: x1.5

    devicestype class-attribute instance-attribute 
    devicestype: Optional[List[str]] = None

    Типы устройств (desktop, mobile). Коэффициент: x1.5

    gender class-attribute instance-attribute 
    gender: Optional[Literal['male', 'female', 'all']] = None

    Пол аудитории. По умолчанию 'all'.

    has_bio class-attribute instance-attribute 
    has_bio: Optional[int] = None

    Наличие Bio (1 - да). Коэффициент: x1.1

    has_fake_check class-attribute instance-attribute 
    has_fake_check: Optional[int] = None

    Прохождение анти-фрод проверки (1 - да). Коэффициент: x1.1

    has_first_name class-attribute instance-attribute 
    has_first_name: Optional[int] = None

    Наличие имени (1 - да). Коэффициент: x1.1

    has_photo class-attribute instance-attribute 
    has_photo: Optional[int] = None

    Наличие фото (1 - да). Коэффициент: x1.1

    has_ru_name class-attribute instance-attribute 
    has_ru_name: Optional[int] = None

    Наличие имени на кириллице (1 - да). Коэффициент: x1.1

    has_username class-attribute instance-attribute 
    has_username: Optional[int] = None

    Наличие юзернейма (1 - да). Коэффициент: x1.1

    languages class-attribute instance-attribute 
    languages: Optional[List[int]] = None

    ID языков (из /filters). По умолчанию [1,2,3,4,5] (RU/UZ/KZ/UA/BY). Null для всех языков.

    old_account class-attribute instance-attribute 
    old_account: Optional[int] = None

    Возраст аккаунта: 1 (>2 лет, x1.1), 2 (>3 лет, x1.2), 3 (>5 лет, x1.3), 4 (>7 лет, x1.4), 5 (>9 лет, x1.5).

    only_premium class-attribute instance-attribute 
    only_premium: Optional[int] = None

    Таргетинг по Premium: 0 - любой (default), 1 - только Premium, 2 - без Premium.

    @field_validator('start_time', 'end_time', mode='before')
@classmethod
def validate_time(cls, v: Optional[Union[str, time]]) -> Optional[Union[str, time]]:
    if isinstance(v, str):
        parts = v.split(':')
        if len(parts) > 0 and len(parts[0]) == 1:
            return f"0{v}"
    return v
